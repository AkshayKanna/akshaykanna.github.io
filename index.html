Tuberculosis Detection Using Chest X-Ray Images

This is my Final Year Project. This involves detection of Tuberculosis from the input chest x-ray images. We obtained the dataset from Kaggle. We train a Convolutional Neural Network (CNN) from the obtained X-Ray images. Also by applying various filters like texture based and Image based we obtain the filtered images. We also use those images for training. By tuning the hyperparameters we able to obtain the Highest Efficency of 91.8%. We also provide a simplified GUI for prediction. User needs to upload the x-ray image. Our model will detect the presence of tuberculosis in the uploaded image with probability percentage. It can generate report in the form of Portable Document Format(pdf).

GUI for Tuning Hyperparamters

This is my Seventh Semester Project. This involves creating a GUI for tuning Hyperparameters in Image Processing. First, user need to login. After he/she can upload dataset and choose hyperparameters like no of Convolutional Layers, Activation Function, No.of hidden layers, No.of Output Layers etc. Once choosen training will be carried out in the server side. Once training completed the user is notified with the Accuracy percentage and the Loss percentage. Also the user can able to download the corresponding weight file and the model file in the form of HDF5 format.
